# Quick Start Guide

## Step 1: Set Up Your Wallet

PerpX Finance runs on **Base Chain**. You'll need a compatible wallet:

- [MetaMask](https://metamask.io) (recommended)
- [Coinbase Wallet](https://www.coinbase.com/wallet)
- Any WalletConnect-compatible wallet

**Add Base Network to MetaMask:**

| Parameter | Value |
| --- | --- |
| Network Name | Base |
| RPC URL | `https://mainnet.base.org` |
| Chain ID | 8453 |
| Currency Symbol | ETH |
| Block Explorer | `https://basescan.org` |

## Step 2: Get USDC on Base

You'll need **USDC** on Base Chain to trade or provide liquidity.

- **Bridge from Ethereum:** Use the [Base Bridge](https://bridge.base.org) to move assets from Ethereum to Base
- **Direct purchase:** Buy USDC through Coinbase and withdraw to Base

## Step 3: Connect to PerpX

1. Visit [https://www.perpx.io](https://www.perpx.io)
2. Click **"Launch App"**
3. Connect your wallet
4. You're ready to trade!

## What Can You Do?

### Trade (Perp DEX)
- Open long/short perpetual positions on supported assets
- Adjustable leverage
- Deposit USDC as margin — no need to hold the underlying asset

### Provide Liquidity
- Deposit USDC or other supported assets into the unified vault
- Receive **PLP tokens** as proof of your liquidity share
- Earn trading fees + protocol incentives

### Stake PPX
- Stake $PPX tokens to earn governance rewards (distributed as esPPX)
- Participate in protocol governance and parameter voting
- 30-day maturity cycle for full reward eligibility
